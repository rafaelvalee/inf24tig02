Descrição das Linguagens

Python

O Python é uma linguagem de programação de alto nível, conhecida pela sua simplicidade e legibilidade...

# Exemplo simples em Python
def saudacao(nome):
    print(f"Olá, {nome}!")
saudacao("Mundo")

HTML

HTML é a linguagem de marcação padrão usada para estruturar o conteúdo de páginas web...

<!DOCTYPE html>
<html>
  <head><title>Exemplo</title></head>
  <body>
    <h1>Olá Mundo</h1>
  </body>
</html>

JavaScript

O JavaScript é a linguagem que dá vida às páginas web. Permite criar interatividade...

// Exemplo de JS
function saudacao(nome) {
  alert("Olá, " + nome + "!");
}
saudacao("Mundo");

XML

O XML é uma linguagem de marcação usada para armazenar e transportar dados estruturados...

<linguagem>
  <nome>Python</nome>
  <dificuldade>Fácil</dificuldade>
</linguagem>
