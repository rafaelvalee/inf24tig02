O que é o JavaScript?

JavaScript é a linguagem de programação que dá vida e interatividade às páginas web. Permite manipular o conteúdo HTML, responder a ações do utilizador, comunicar com servidores e criar aplicações modernas.

Como inserir JavaScript numa página


<script>
  alert("Olá, JavaScript!");
</script>

Recomenda-se colocar o <script> no final do <body> ou usar defer para garantir que o HTML já foi carregado.


<script src="ficheiro.js"></script>
<script>alert("Oi")</script>
<javascript>alert("Oi")</javascript>


Variáveis e tipos de dados


let nome = "Rafael";
const idade = 19;
var altura = 1.80;

console.log(typeof idade); // "number"
  • let: variável que pode mudar de valor.
  • const: variável constante (não pode ser alterada).
  • var: forma antiga, evita-se usar atualmente.
  • Principais tipos: string, number, boolean, object, array, null, undefined.

Funções e arrow functions


// Função tradicional
function soma(a, b) {
  return a + b;
}

// Arrow function
const multiplicar = (a, b) => a * b;

Arrow function não pode ter parâmetros
Arrow function não tem o seu próprio this
Arrow function só pode retornar números

Boas práticas & Dicas rápidas

  • Usa sempre let e const em vez de var.
  • Evita variáveis globais desnecessárias.
  • Usa sempre console.log() para debugar e experimentar.
  • Separa código JS em ficheiros externos para organização.
  • Valida o teu JS em JSHint ou ESLint.
