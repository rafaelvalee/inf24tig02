O que é o HTML?

HTML (HyperText Markup Language) é a linguagem usada para estruturar o conteúdo das páginas web. Não é uma linguagem de programação, mas sim de marcação.

Estrutura de um documento HTML


<!DOCTYPE html>
<html lang="pt">
  <head>
    <meta charset="UTF-8">
    <title>Exemplo</title>
  </head>
  <body>
    <h1>Olá, mundo!</h1>
  </body>
</html>
  • <!DOCTYPE html> – define que o documento é HTML5.
  • <head> – metadados, título, links para CSS/JS.
  • <body> – o conteúdo visível na página.

Elementos, atributos e semântica

O HTML é composto por elementos (tags) e pode ter atributos que fornecem informações extra.
A escolha correta de elementos semânticos ajuda o browser e leitores de ecrã a entender melhor o conteúdo. 


<article class="noticia" data-id="27">
  <h2>Novidade</h2>
  <p>Saiu uma nova versão do HTML!</p>
</article>
  • O atributo class serve para aplicar estilos (CSS).
  • O atributo data-id é um atributo personalizado (útil para JS).

Listas, tabelas e imagens

Para organizar informação, podes usar listas ordenadas, não ordenadas ou de definições. 


<ul>
  <li>Python</li>
  <li>JavaScript</li>
  <li>HTML</li>
</ul>

Tabelas devem ter cabeçalho, corpo e rodapé: 


<table>
  <thead><tr><th>Linguagem</th><th>Dificuldade</th></tr></thead>
  <tbody>
    <tr><td>Python</td><td>Fácil</td></tr>
    <tr><td>JavaScript</td><td>Média</td></tr>
  </tbody>
  <tfoot><tr><td colspan="2">Exemplo</td></tr></tfoot>
</table>

Formulários

Formulários permitem recolher dados dos utilizadores. Usa sempre <label> para acessibilidade. 


<form action="#" method="post">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome">
  <input type="submit" value="Enviar">
</form>

Boas práticas & Dicas rápidas

  • Usa sempre elementos semânticos para melhorar SEO e acessibilidade.
  • Coloca o lang="pt" na tag <html> para definir o idioma.
  • Valida o teu HTML no W3C Validator.
  • O HTML não é sensível a maiúsculas/minúsculas, mas usa sempre minúsculas por convenção.
