O que é o Python?

Python é uma linguagem de programação de alto nível, famosa pela sua simplicidade e legibilidade. É utilizada em áreas como web, automação, ciência de dados, inteligência artificial, scripts, etc.

Sintaxe e variáveis

Em Python não precisas declarar tipos de variáveis. Exemplo:


nome = "Rafael"
idade = 19
altura = 1.78
  • O tipo de cada variável é detetado automaticamente.
  • Podes mudar o valor e o tipo da variável em qualquer altura.

x = 5
int x = 5
x=5


Tipos de dados básicos

  • int – Números inteiros (ex: 1, -23)
  • float – Números decimais (ex: 2.5, -0.1)
  • str – Texto (ex: "Olá")
  • bool – Booleano (True / False)
  • list – Lista de elementos (ex: [1, 2, 3])

Tratamento de erros (try/except)

Se escreveres código que pode falhar, usa try/except:


try:
    n = int(input("Número: "))
    print(10 / n)
except ZeroDivisionError:
    print("Não podes dividir por zero!")
except ValueError:
    print("Tens de escrever um número válido!")

Dicas úteis

  • Podes instalar pacotes externos com pip (ex: pip install requests).
  • Indentação é obrigatória em Python. Não uses tab misturado com espaços.
  • Comentários começam com #.
  • O Python é "case sensitive": Nome ≠ nome.
